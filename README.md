# LeagueStats

## Global dependencies
```
Vue CLI
Adonis CLI
MongoDB
```

## Project setup
```
cd client
npm install

cd server
npm install
cp .env.example .env
adonis migration:run
```

### Compiles and hot-reloads for development
```
cd client
npm run serve
```

### Compiles and minifies for production
```
cd client
npm run build
```

### Run NodeJS server
```
cd server
adonis serve --dev
```
